# twinkly-animator

To use:

```
npm install
```

For the marquee scroller:

```
node mydemo4.js Twinkly-IP text
```

to simulate

```
SIM=1 node mydemo4.js Twinkly-IP text
```
eg.

```
node mydemo4.js 192.168.2.217 "GARY SUCKS"
```
